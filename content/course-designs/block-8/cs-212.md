## CS-212 — Data Analysis & Responsible AI

**1 credit**

### Working description

Notebook-based data analysis as the CULMINATION of the STAT-101 statistics foundation and the data-visualization throughline (B2/B4/B5/B7) — integrative, not new-from-scratch. Honest visualization and data ethics. The genuinely new skill: high-stakes verification and critique of AI-generated code for correctness and security (AI bounded-practice final pass; prerequisite for the Year 3–4 Agentic AI specialization, which this course precedes but does not duplicate).

## Competency Assessments

| Assessment type | This pass | Competencies evidenced |
|---|---|---|
| Data Pipeline Task | Notebook pipeline with data provenance and ethical documentation | Manage Data Resources |
| Data Investigation | Full notebook investigation with visualization, ethical framing, and honest uncertainty reporting | Investigate Questions with Data |
| Data Communication Task | Honest visualization with ethical documentation — what does this chart claim, is the claim justified? | Communicate Data Insights |
| Ethical Evaluation | AI-assisted analysis critique — evaluate an AI-generated artifact for correctness, bias, security risk, and responsible use | Evaluate Ethical Implications |
| AI-Assisted Practice Task | AI code critique — evaluate AI-generated code for correctness, security vulnerabilities, and bias | Utilize AI Responsibly |
